RESEND EMAIL NOT WORKING - FIX
================================

The Resend API is working (test confirmed), but Django isn't using it.

SOLUTION:
---------

1. RESTART Django Server
   - Stop the server (Ctrl+C)
   - Start it again: python manage.py runserver
   
   This is REQUIRED because .env changes need server restart!

2. Test Forgot Password
   - Go to: http://127.0.0.1:8000/forgot-password/
   - Enter department email
   - Check console logs

3. Look for These Messages:
   [Email] DEBUG: Resend API Key present: True
   [Email] DEBUG: API Key starts with: re_dcPXW18...
   [Email] Attempting to send via Resend API to user@example.com
   [Email] Resend API: Email sent successfully

4. If Still Using SMTP:
   Check console for:
   [Email] DEBUG: Resend API Key present: False
   
   This means .env not loaded. Solution:
   - Make sure .env file is in project root
   - Restart server completely
   - Check .env has: RESEND_API_KEY=re_dcPXW18d_9WjpDnmYLKCKn2xDzFAWNW2d

QUICK TEST:
-----------
Run this to verify settings are loaded:

python -c "import os; os.environ.setdefault('DJANGO_SETTINGS_MODULE', 'smartcity.settings'); import django; django.setup(); from django.conf import settings; print('RESEND_API_KEY:', getattr(settings, 'RESEND_API_KEY', 'NOT FOUND'))"

Should show: RESEND_API_KEY: re_dcPXW18d_9WjpDnmYLKCKn2xDzFAWNW2d

If it shows "NOT FOUND", the .env file isn't being loaded properly.
